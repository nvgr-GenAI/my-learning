# Sorting Algorithms

Welcome to the Sorting Algorithms section. Learn about different algorithms to sort data efficiently.

## Sorting Algorithms

- **[Bubble Sort](bubble-sort.md)** - Simple comparison-based sort
- **[Selection Sort](selection-sort.md)** - Finds minimum/maximum iteratively
- **[Insertion Sort](insertion-sort.md)** - Builds sorted array one item at a time
- **[Quick Sort](quick-sort.md)** - Efficient divide-and-conquer sort
- **[Merge Sort](merge-sort.md)** - Stable divide-and-conquer sort
- **[Heap Sort](heap-sort.md)** - Uses heap data structure

## Comparison

| Algorithm | Best | Average | Worst | Space | Stable |
|-----------|------|---------|-------|-------|--------|
| Bubble | O(n) | O(n²) | O(n²) | O(1) | Yes |
| Selection | O(n²) | O(n²) | O(n²) | O(1) | No |
| Insertion | O(n) | O(n²) | O(n²) | O(1) | Yes |
| Quick | O(n log n) | O(n log n) | O(n²) | O(log n) | No |
| Merge | O(n log n) | O(n log n) | O(n log n) | O(n) | Yes |
| Heap | O(n log n) | O(n log n) | O(n log n) | O(1) | No |
