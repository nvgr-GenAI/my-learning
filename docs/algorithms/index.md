=== "Problem-Solving Approach"
    **5-Step Framework:** Understand ‚Üí Plan ‚Üí Implement ‚Üí Optimize ‚Üí Review

    **Pattern Recognition Guide:**

    | Problem Type | Data Structure/Approach |
    |--------------|------------------------|
    | Multiple conditions | Hash table or set |
    | Sorted array | Binary search |
    | Need ordering | Array or linked list |
    | Unique elements only | Hash set |
    | Continuous min/max | Heap (priority queue) |
    | Tree/graph traversal | BFS (level order) or DFS |
    | Overlapping subproblems | Dynamic programming |
